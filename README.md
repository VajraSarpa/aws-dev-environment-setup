# AWS Development Environment Setup

Complete guide and scripts for setting up a cloud-based development environment on AWS EC2, accessible from a 2012 MacBook Pro (or any computer) via VS Code Remote-SSH.

## What's Included

- Step-by-step AWS EC2 instance setup
- SSH configuration for seamless connections
- VS Code Remote-SSH integration
- Installation scripts for Python, Node.js, Go, Rust, Java, Docker
- Cost management strategies
- Daily usage workflows
- Troubleshooting guide

## Quick Start

See [aws-dev-environment-setup.md](aws-dev-environment-setup.md) for the complete guide.
